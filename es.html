Características

Personaliza los comandos de instalación

Puedes utilizar las siguientes 2 variables especiales para componer tu comando personal:

  • <package>El nombre del paquete npm
  • <version>Versión seleccionada actualmente

Y luego visita cualquier página de paquete en npmjs.com para ver el resultado. ✨️

Ejemplos personalizados

Aquí hay algunos ejemplos que puedes usar en Versatile Npm:

  • Instalar como dependencia de desarrollo

    Plantilla
    npm i -D <package>
    Salidanpm i -D lodash

  • Usar Yarn

    Plantilla
    yarn add <package>
    Salidayarn add lodash

  • Usar PNPM

    Plantilla
    pnpm i <package>
    Salidapnpm i lodash

  • Usar Bun

    Plantilla
    bun install <package>
    Salidabun install lodash

  • Instalar una versión especificada

    Plantilla
    npm install <package>@<version>
    Salidanpm install lodash@4.17.21

  • Instalar definiciones de tipo

    Plantilla
    npm i -D @types/<package>
    Salidanpm i -D @types/lodash

  • Eliminar un paquete

    Plantilla
    npm uninstall <package>
    Salidanpm uninstall lodash

  • Muestra tu amor por un paquete 🫶

    Plantilla
    I love <package>
    SalidaI love lodash
